Teating_GTN_data

This repository contains the data and code corresponding to the figures and analyses in the manuscript.

GHZ_LR.m – Code for Fig. 3(a) in the main text.
W_LR.m – Code for Fig. 4(a) in the main text.
shuangy1.m – Code for Fig. 3(b) in the main text.
shuangy2.m – Code for Fig. 4(b) in the main text.
SS_F.m – Code for the appendix section Measured correlators of the Svetlichny inequality.
trc.m – Code for GHZ state tomography.
trcw.m – Code for W state tomography.
BBar_3T_R.m – Code for the real part of the density matrix.
BBar_3T_I.m – Code for the imaginary part of the density matrix.
NOISE/ – Folder containing all noisy state data and related code.
